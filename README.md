# Kanchwala Jewellers Demo

Luxury jewellery storefront demo built as a static website.

## Repository

- GitHub: [abizertc-arch/Kanchwala](https://github.com/abizertc-arch/Kanchwala)

## Live Demo

- GitHub Pages: [https://abizertc-arch.github.io/Kanchwala/](https://abizertc-arch.github.io/Kanchwala/)

If the demo does not open immediately, GitHub Pages may still be publishing the site from the `main` branch.

## Project Files

- `index.html` - homepage
- `collection.html` - collection listing page
- `product.html` - product detail page
- `styles.css` - shared styling
- `cart.js` - cart and checkout UI behavior
- `images/` - jewellery product images and visual assets

## Local Preview

Because this is a static site, you can preview it in either of these ways:

1. Open `index.html` directly in a browser.
2. Serve the folder with any static server, then open the local URL in your browser.

## Deployment

The repository is configured to publish with GitHub Pages from:

- Branch: `main`
- Folder: `/ (root)`

## Notes

- This project is a frontend demo and does not include a backend checkout flow.
- Product pricing and content are demo content for presentation purposes.
